# Фотографии продуктов

Эта папка содержит изображения продуктов AERONIX.

## Структура файлов

Для каждого продукта нужно добавить:
- `{product-id}-main.jpg` - главное изображение для карточки (рекомендуемый размер: 800x800px)
- `{product-id}-1.jpg` - фото 1 для галереи в модальном окне (рекомендуемый размер: 1200x800px)
- `{product-id}-2.jpg` - фото 2 для галереи
- `{product-id}-3.jpg` - фото 3 для галереи

### Пример для продукта ДК:
```
products/
  dk-main.jpg      # Основное фото клапана ДК
  dk-1.jpg         # Фото клапана ДК на радиаторе
  dk-2.jpg         # Детальное фото клапана ДК
  dk-3.jpg         # Фото установки клапана ДК
```

   - dk-main.jpg
   - dk-1.jpg, dk-2.jpg, dk-3.jpg

2. **ТК** (AERONIX-TK) - для тёплых помещений

3. **БК-100** (AERONIX-БК-100) - для холодных помещений

4. **БК-40** (AERONIX-БК-40) - для холодных помещений
   - bk40-main.jpg
   - bk40-1.jpg, bk40-2.jpg, bk40-3.jpg

## Требования к изображениям

- **Формат**: JPG или PNG
- **Размер главного фото**: 800x800px (квадрат) или 4:3
- **Размер фото галереи**: 1200x800px (16:9 или 4:3)
- **Вес файла**: не более 500KB для оптимизации загрузки
- **Фон**: желательно белый или нейтральный для главных фото
- **Качество**: высокое, без размытия

## Референсы

Используйте приложенные фотографии как пример:
- Белый круглый клапан на стене
- Клапан установленный на радиаторе
- Детальные снимки компонентов

## Fallback

Если фото отсутствует, автоматически отобразится placeholder с иконкой Wind.
